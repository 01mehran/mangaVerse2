# 📚 MangaVerse

A modern manga discovery application built with React and Tailwindcss, powered by the Jikan API.

## ✨ Features

- 🔍 Search manga by title
- 📖 View detailed manga information
- ⭐ Display ratings, rankings, and popularity
- 📄 Read manga synopses and backgrounds
- 📱 Fully responsive design
- ⚡ Fast and modern user experience
- 🎯 Compare multiple manga and discover recommendations
- 💾 Persistent data using Local Storage

## 🧠 Compare & Recommend Feature

One of MangaVerse's standout features is the manga comparison system.

Users can:

1. Select multiple manga.
2. Compare their statistics.
3. Analyze genre overlap.
4. Generate personalized recommendations based on selected manga.

This helps users discover new manga that match their interests.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/01mehran/mangaVerse2.git
```

Navigate to the project folder:

```bash
cd mangaVerse2
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
